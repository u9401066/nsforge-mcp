## 臨床背景

某些藥物（如 **Phenytoin**、高劑量 **Aspirin**、**Ethanol**）的代謝酶會飽和，呈現**非線性藥物動力學**。這意味著：

- 劑量增加一點點 → 血中濃度可能**暴增**
- 治療範圍窄，容易中毒

當病人有**發燒**或**低體溫**時，酵素動力學參數會改變，更難預測濃度變化。

---

## 基礎公式

### Michaelis-Menten 方程

$$Rate = \frac{V_{max} \cdot C}{K_m + C}$$

- $V_{max}$：最大代謝速率 (mg/L/h)
- $K_m$：米氏常數，達到 $0.5 V_{max}$ 時的濃度 (mg/L)
- $C$：藥物濃度 (mg/L)

### 溫度依賴性

**$V_{max}$ 的 Arrhenius 方程**：
$$V_{max}(T) = V_{max,ref} \cdot e^{\frac{E_a}{R}\left(\frac{1}{T_{ref}} - \frac{1}{T}\right)}$$

**$K_m$ 的 van't Hoff 方程**：
$$K_m(T) = K_{m,ref} \cdot e^{\frac{\Delta H_m}{R}\left(\frac{1}{T_{ref}} - \frac{1}{T}\right)}$$

---

## 最終推導結果

$$\boxed{Rate(T) = \frac{C \cdot V_{max,ref} \cdot e^{\frac{E_a}{R}\left(\frac{1}{T_{ref}} - \frac{1}{T}\right)}}{C + K_{m,ref} \cdot e^{\frac{\Delta H_m}{R}\left(\frac{1}{T_{ref}} - \frac{1}{T}\right)}}}$$

### SymPy 表達式

```python
C*V_max_ref*exp(E_a*(1/T_ref - 1/T)/R)/(C + K_m_ref*exp(Delta_H_m*(1/T_ref - 1/T)/R))
```

## 臨床應用

### 使用時機

1. **發燒病人的 Phenytoin TDM**
   - 預期清除率增加 → 濃度可能下降
   - 可能需要暫時增加劑量

2. **低體溫治療**
   - 代謝減慢 → 濃度可能升高
   - 需監測毒性

3. **高劑量 Aspirin**
   - 抗風濕劑量時呈飽和動力學
   - 發燒時更難預測

## 假設條件

1. 酵素遵循 Michaelis-Menten 動力學
2. $V_{max}$ 的溫度依賴遵循 Arrhenius 方程
3. $K_m$ 的溫度依賴遵循 van't Hoff 方程
4. 單一代謝途徑主導
5. 酵素結構在溫度範圍內穩定

## 使用限制

1. 僅適用於 32-42°C（超出範圍酵素可能變性）
2. 假設單一酵素途徑
3. 不考慮溫度對蛋白結合的影響
4. 需要實驗測定每種藥物的 $E_a$ 和 $\Delta H_m$

## SymPy-MCP 驗證代碼

```python
# 定義變數
from sympy import symbols, exp, simplify

C, V_max, K_m = symbols('C V_max K_m', positive=True)
V_max_ref, K_m_ref = symbols('V_max_ref K_m_ref', positive=True)
T, T_ref, E_a, Delta_H_m, R = symbols('T T_ref E_a Delta_H_m R', positive=True)

# 基礎 Michaelis-Menten
mm_rate = (V_max * C) / (K_m + C)

# 溫度校正
vmax_temp = V_max_ref * exp(E_a / R * (1/T_ref - 1/T))
km_temp = K_m_ref * exp(Delta_H_m / R * (1/T_ref - 1/T))

# 代入
rate_temp = mm_rate.subs(V_max, vmax_temp).subs(K_m, km_temp)
print(rate_temp)
```
