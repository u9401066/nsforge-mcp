# Project Brief

> 📌 此檔案描述專案的高層級目標和範圍，建立後很少更改。

## 🎯 專案目的

提供一個整合 AI 輔助開發工具的專案模板，包含：
- 憲法-子法規則系統
- Claude Skills 模組化技能
- Memory Bank 跨對話記憶
- DDD + DAL 架構指引

## 👥 目標用戶

- 使用 VS Code + GitHub Copilot 的開發者
- 希望建立 AI 輔助開發工作流的團隊
- 想要標準化專案結構的開發者

## 🏆 成功指標

- [ ] 專案能作為 GitHub Template 使用
- [ ] Skills 能正確觸發和執行
- [ ] Memory Bank 能跨對話保持記憶
- [ ] CI/CD 流程能正常運作

## 🚫 範圍限制

- 此模板不包含實際業務邏輯
- 不提供特定框架實作（僅提供結構指引）
- 不包含生產環境部署配置
